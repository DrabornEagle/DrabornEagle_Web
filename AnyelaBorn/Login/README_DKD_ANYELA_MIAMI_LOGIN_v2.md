# DKD AnyelaBorn Miami Login v2

Bu paket `/AnyelaBorn/Login` altında Miami temalı login ekranını çalıştırır.

- `/AnyelaBorn/` ana route şimdilik pasif hazırlık ekranıdır.
- `/AnyelaBorn/Login/` login ve üye ol arayüzüdür.
- Sosyal medya ikonları modern SVG/CSS ikonlarla yenilendi.
- Login penceresi mobilde daha geniş hale getirildi.

Gerçek Google/Apple/Instagram auth ve backend login bağlantısı henüz eklenmedi.
