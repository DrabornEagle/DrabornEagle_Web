# Anyela Born Club Mobile React

Bu paket, `www.draborneagle.com/AnyelaBorn` için mobil öncelikli React/Vite arayüzüdür.

## İçerik

- Ana Sayfa
- Paketler ve Deneyimler
- Özel Sohbet Odası
- Ödeme ve Başlangıç
- Reklam İş Birliği
- Voice Deneyimi
- Favoriler
- Hesabım

Kadın görselleri kullanıcının yüklediği görsellerden `src/assets` klasörüne eklenmiştir. İkonlar lucide-react kütüphanesiyle kullanılmıştır; inline SVG dosyası eklenmemiştir.

## Kurulum

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notlar

- IBAN bilgileri örnek/placeholder olarak bırakılmıştır.
- Bu sürüm ön yüz tasarım prototipidir; ödeme onayı, gerçek chat, dosya yükleme ve admin panel entegrasyonu için backend bağlanmalıdır.
- Tasarım 9:16 mobil görünüm için optimize edilmiştir.
