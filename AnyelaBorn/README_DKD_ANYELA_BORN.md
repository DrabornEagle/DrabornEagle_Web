# Anyela Born Club Mobile Web Rebuild

Bu klasör `https://www.draborneagle.com/AnyelaBorn/` rotası için sıfırdan hazırlanmış mobil uyumlu Anyela Born Club arayüzüdür.

## İçerik

- `index.html`: React tabanlı tek sayfa uygulama. Vite kullanılmaz.
- `assets/img/`: Kullanıcının gönderdiği Anyela görselleri.
- Sayfalar: Ana Sayfa, Giriş/Kayıt, Paketler, Fan Sohbet, Voice Chat, Özel Görsel & Video, İşletme Reklam, Ödeme/Dekont, Özel Chat Odası, Hesabım/Siparişlerim, Admin Panel.

## Not

İkonlar manuel SVG olarak çizilmedi. React Icons kütüphanesi CDN üzerinden kullanılır. Site GitHub Pages/static hosting altında doğrudan çalışacak şekilde hazırlanmıştır.
