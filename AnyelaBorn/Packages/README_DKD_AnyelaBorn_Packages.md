# DKD Anyela Born Packages Page

Bu paket, `DrabornEagle_Web` reposunda `https://www.draborneagle.com/AnyelaBorn/Packages/` sayfasını güncellemek için hazırlandı.

## İçerik

- `AnyelaBorn/Packages/index.html`
  - Miami temalı paketler ekranı
  - Gönderilen 864x1536 orijinal paket görseli
  - Paket seçimine göre detay/dakika/içerik açıklaması
  - IBAN ödeme bilgisi
  - Dekont seçme alanı
  - Teklif Oluştur hesaplayıcısı
- `AnyelaBorn/Packages/dkd_anyela_packages_original_20260530.jpg`
  - Kullanıcının gönderdiği orijinal JPEG dosyası birebir kopyalandı, yeniden sıkıştırılmadı.

## Not

Bu teslimat statik web MVP ekranıdır. Form, dekont dosyasını tarayıcı tarafında seçtirir ve müşteriye onay mesajı gösterir. Dekontun gerçekten sunucuya yüklenmesi, admin panelde görünmesi ve onaydan sonra chat odasının açılması için bir sonraki adımda Supabase Storage + database + admin panel bağlantısı kurulmalıdır.

## dkd_ standardı

Yeni HTML, CSS ve JavaScript içindeki yeni/özel sınıf, id, fonksiyon ve değişken adları `dkd_` / `dkd` standardıyla yazıldı.
