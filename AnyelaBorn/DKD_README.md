# DKD Anyela Born Club Premium React v1.3

Bu paket `www.draborneagle.com/AnyelaBorn/` için hazırlanmış mobil öncelikli, premium React tasarım sürümüdür.

## Önemli
- Yayına alınan çalışma dosyaları `index.html` + `assets/dkd-anyela-born-v13.css` + `assets/dkd-anyela-born-v13.js` içindedir.
- Canlı sürüm React ile çalışır; `assets/dkd-anyela-born-v13.js` React uygulamasını mount eder.
- `src/` klasörü profesyonel React + TypeScript kaynak yapısı olarak bırakılmıştır.
- Termux scripti npm build çalıştırmaz; Vite hatasına takılmamak için hazır statik React sürümünü direkt push eder.

## Satış v1 mantığı
- IBAN + dekont manuel onay.
- Sohbet / ses / görsel / video ilk aşamada manuel-yarı manuel teslim.
- Tam otomatik chat sonraki sürümdür.
