DKD Anyela Born Club — Miami React Mobile Page

Bu paket, DrabornEagle_Web reposundaki /AnyelaBorn/ içeriğini temizleyip yerine mobil-first React tabanlı yeni Miami temalı Anyela Born Club ana sayfası koymak için hazırlandı.

İçerik:
- AnyelaBorn/index.html
- AnyelaBorn/dkd_anyela_born_app.js
- AnyelaBorn/dkd_anyela_born_styles.css

Notlar:
- Sayfa React 18 UMD ile çalışır. GitHub Pages / statik hosting üzerinde doğrudan /AnyelaBorn/ olarak açılır.
- Tasarım mobil-first hazırlanmıştır. Masaüstünde telefon mockup çerçevesi gibi ortalanır.
- Tüm ana CSS sınıfları ve JS component fonksiyonları dkd_ / dkd- standardına göre adlandırılmıştır.
- Görsel, video, ödeme ve chat butonları şu an UI mockup/landing davranışındadır. Backend/Supabase bağlantıları sonraki aşamada eklenmelidir.
