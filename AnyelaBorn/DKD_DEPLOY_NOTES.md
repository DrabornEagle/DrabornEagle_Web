# DKD AnyelaBorn Premium React v1.2

Kaynak dili: React + TypeScript.

Bu klasörde GitHub Pages için derlenmiş statik çıktı kökte hazırdır:
- `index.html`
- `assets/dkd-anyela-born-v12.js`
