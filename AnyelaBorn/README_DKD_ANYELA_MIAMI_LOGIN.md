# DKD AnyelaBorn Miami Login v1

Bu klasör `https://www.draborneagle.com/AnyelaBorn/` için hazırlanmış statik, mobil uyumlu Miami temalı Anyela Born Club giriş/üye ol sayfasıdır.

## Dosyalar

- `index.html` — yayına alınacak ana sayfa
- `dkd_anyela_miami_login.css` — mobil/desktop uyumlu tasarım
- `dkd_anyela_miami_login.js` — giriş/üye ol sekmesi, şifre göster/gizle ve demo buton davranışları
- `assets/dkd_anyela_miami_hero.jpg` — gönderilen referans görselden kırpılan Miami hero alanı
- `.nojekyll` — GitHub Pages statik dosya uyumluluğu

## Not

Sosyal giriş ve gerçek kullanıcı oturumu için backend bağlantısı henüz bağlanmadı. Bu sürüm görsel/login arayüzünü hazırlar.
