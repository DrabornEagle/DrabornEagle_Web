# AnyelaBorn/Packages

Bu klasörü sitenin statik yayın köküne koyduğunda sayfa şu adreste açılır:

`https://www.draborneagle.com/AnyelaBorn/Packages/`

Vite/React projesinde yayın kökü genelde `public/` olduğu için kurulum scripti `public` klasörü varsa buraya kopyalar:

`public/AnyelaBorn/Packages/`

Düz GitHub Pages/static hosting kullanıyorsan kök klasördeki şu yapı yeterlidir:

`AnyelaBorn/Packages/index.html`
