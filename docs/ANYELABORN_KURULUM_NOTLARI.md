# Anyela Born Club Miami Mobile v1

Bu paket iki kullanım sağlar:

1. `AnyelaBorn/index.html`  
   Direkt `https://www.draborneagle.com/AnyelaBorn/` yolunda çalışabilecek bağımsız mobil web sayfasıdır.

2. `src/anyelaborn/DkdAnyelaBornPage.jsx` + `src/anyelaborn/dkd-anyela-born.css`  
   React projesine component olarak eklenebilecek sürümdür.

## İçerik

- Miami temalı mobil ana sayfa
- Türkçe arayüz
- Paket kartları
- IBAN/dekont alanı
- Kurallar alanı
- Mobil alt menü
- React component yapısına uygun `dkd_` isimlendirme

## Sonraki teknik adım

- Dekont yükleme butonu Supabase Storage veya mevcut backend upload endpointine bağlanacak.
- Paket seç butonları sipariş oluşturma ekranına bağlanacak.
- Özel oda butonu chat odasına bağlanacak.
