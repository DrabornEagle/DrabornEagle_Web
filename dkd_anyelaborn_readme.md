# AnyelaBorn Home Update v1

Bu paket `https://www.draborneagle.com/AnyelaBorn/Home` için hazırlanmıştır.

## İçerik

- `AnyelaBorn/Home/index.html`
  - Gönderilen ana görsel kalite bozulmadan kullanıldı.
  - Profilim, Sohbet Odası, Paketler ve Markan için Anyela alanları tıklanabilir şeffaf hotspot olarak eklendi.
- `AnyelaBorn/Profile/index.html`
  - Ayrı, detaylı Miami temalı profil ekranı.
- `AnyelaBorn/Chat/index.html`
  - Ayrı, detaylı Miami temalı paket odaklı sohbet odası demo ekranı.
- `AnyelaBorn/assets/dkd_anyela_home_miami_original.jpg`
  - Orijinal görsel byte-byte kopyalandı, yeniden sıkıştırma yapılmadı.
- `AnyelaBorn/assets/dkd_anyela_miami_pages.css`
  - Profil ve sohbet odası stilleri.
- `AnyelaBorn/assets/dkd_anyela_chat_room.js`
  - Demo sohbet etkileşimi.

## Yönlendirme

- Profilim → `/AnyelaBorn/Profile/`
- Sohbet Odası → `/AnyelaBorn/Chat/`
- Paketler → `https://www.draborneagle.com/AnyelaBorn/Packages/`
- Markan için Anyela → şimdilik pasif bırakıldı.
